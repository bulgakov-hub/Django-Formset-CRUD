### Install

Clone a repository
```bash
$ git clone https://github.com/bulgakov-hub/Django-Formset-CRUD.git
```
Create a virtual environment
```bash
$ python3 -m venv venv
```
Activate the virtual environment 
```bash
$ source venv/bin/activate
```
Install requirements
```bash
$ pip install -r requirements.txt
```
Migrations
```bash
$ ./manage.py migrate
```
Run server
```bash
$ ./manage.py runserver
```
